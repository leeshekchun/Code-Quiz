# Code-Quiz


## Link to the application: https://leeshekchun.github.io/Code-Quiz/


## There are 3 questions in the quiz. Each correct answer will gain you +1 point, and each wrong answer will deduct 10 seconds on the timer. 
## When you answered all the questions or when the timer goes off, the test will end.

## In order to find the highest score, the quiz needs to be tested twice.
